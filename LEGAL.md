## 2. Fair Use & Open Data Context
This tool acts solely as an accessibility wrapper and text utility to help citizens read their publicly distributed local draft electoral rolls. The software does not scrape hidden databases, alter official government systems, or claim ownership over public records.

## 3. Extreme Privacy Safeguards (Zero-PII Abuse)
Contributors and operators of this open-source codebase agree to the following strict operating guardrails:
* **No Public Scraping Databases:** This application must never be hosted as a public, unauthenticated master lookup database of voter details.
* **Encryption Mandatory:** Any personal data handled for notifications must be structured via the built-in AES-256 and HMAC-SHA256 security utilities.
